# Coffee Collection 

Coffee Collection is a simple app for coffee lovers. Enjoy in exploring coffees from all around the world and give it a review. User can create own personal coffee collection as well.

This project is built on Ruby on Rails with JavaScript


# Installation

To install this application use the following steps:
  * Fork and clone this repository, and then execute:
  ```  
    $ bundle install
    $ rake db:migrate
    $ rake db:seed

  ```

  * Then run:
  ```
    $ rails s
  ```
  * Open up a new browser window and navigate to:
  ```
    localhost:3000
  ```
